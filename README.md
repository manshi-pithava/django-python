# 🛒 Django E-Shop

A full-featured eCommerce web application built using Django.

---

## 🚀 Features

- 👤 User Registration & Login  
- 🛍 Product Listing  
- 🔍 Product Search  
- 🛒 Add to Cart  
- 📦 Order Management  
- 🧑‍💼 Admin Dashboard  

---

## 🛠 Tech Stack

- Python 🐍  
- Django 🌐  
- HTML, CSS, Bootstrap 🎨  
- SQLite 🗄  

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
git clone https://github.com/manshi-pithava/django-python.git
cd django-python
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
